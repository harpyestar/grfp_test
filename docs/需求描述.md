
# === 3:导出邀约酒店名单、导入集团意向酒店名单增加酒店ID ===
## 需求描述
需求标题：导出邀约酒店名单、导入集团意向酒店名单增加酒店ID
具体描述：
1. 使用一个存在指定存在酒店的项目，进入到该签约的修改项目编辑页面
2. 在邀请酒店栏中普通单店的页面下，通过数据库的表获取数据：grfp.t_project_intent_hotel
   - 这是mysql数据库的表，需要根据project_id和invite_status=1，grfp.t_hotel表中的t.hotel_id, 获取酒店名t.name_en_us
3. 另外需要在集团意向单店页面中，通过数据库的表获取数据：grfp.t_project_invite_hotel
   - 这是mysql数据库的表，需要根据project_id，并且需要联合grfp.t_hotel表中的t.hotel_id, 获取酒店名t.name_en_us
4. 实现代码在operate的rfp_management目录下，使用已有用例文件：test_edit_rfp_project_tabs.py，新构建一个用例方法
   - tests/operate/rfp_management/test_edit_rfp_project_tabs.py
5. 注意测试用例层和page形式写法，做到POM形式，元素方法和定位形式分明
    - page在：pages/operate/rfp_management/edit_rfp_project_page.py
6. 使用参数化的形式提取出来，因为里面的传入项目名可能会改变
7. 下方涉及到数据连接的，希望提取成公共类放到utils中，涉及相关查询的操作等

## 步骤描述
1. 首先到/home页
2. 先确定参数化是英文还是中文
3. 然后先右上角设置语言
4. 点击RFP Management菜单，选择下拉菜单中的：Contracting选项 
5. 然后再Contracting的页面中，需要先选择Started的Tab项
6. 点击搜索栏目的Project填入固定项目名：自动化测试项目-价格页签固定酒店项
7. 点击搜索按钮，精准匹配出现的首项
8. 接着在首项目的Operation栏中，存在编辑按钮：Modify Project，可能存在多个需要精准匹配首个
9. 连接Mysql数据，查找信息，缓存下来
   - 具体的数据连接信息：ip:172.16.88.71, 端口:3306, 用户名：root, 密码：ABcd@1234
10. 其中数据库查询的设计，涉及普通酒店信息（grfp.t_project_intent_hotel这个相关sql查询到的）、集团酒店信息（grfp.t_project_invite_hotel这个相关sql查询到的）
11. 在修改编辑页面中，选择【邀请酒店】Tab：Invited Hotel
12. 点击导出按钮：Download Template，这里导出的是：普通酒店信息
13. 断言：普通酒店信息的Excel中是否存在酒店ID字段，另外对比每一行酒店Id和酒店名，对比在数据库查询表的结果
14. 再点击进入【添加酒店集团意向单店】按钮：Add preferred hotels group
15. 再次点击导出按钮：Export, 这里导出的是：集团酒店信息
16. 断言：集团酒店信息的Excel中是否存在酒店ID字段，另外对比每一行酒店Id和酒店名，对比在数据库查询表的结果

def run(playwright: Playwright) -> None:
    browser = playwright.chromium.launch(headless=False)
    context = browser.new_context()
    page.get_by_role("button", name="RFP Management").click()
    page.get_by_role("menuitem", name="Contracting").click()
    page.get_by_role("tab", name="Started", exact=True).click()
    page.locator("#el-id-1615-59").click()
    page.locator("#el-id-1615-59").fill("自动化测试项目-价格页签固定酒店项")
    page.locator(".search > .btn").click()
    page.get_by_text("Modify Project").first.click()
    page.get_by_role("tab", name="Invited Hotel").click()
    with page.expect_download() as download8_info:
        page.get_by_text("Download Template").click()
    download8 = download8_info.value
    page.get_by_role("button", name="Add preferred hotels group").click()
    with page.expect_download() as download9_info:
        page.get_by_text("Export").click()
    download9 = download9_info.value
