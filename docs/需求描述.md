## 需求描述
需求标题：【GRFP】同时有LRA NLRA报价中签时提示的选项 TRUN ON 有重复选项
具体描述：
1. 使用一个存在项目，进去点击修改项目;
2. 在修改项目基础信息页中，检测NLRA和LRA开关数量
3. 实现代码在operate的rfp_management目录下，沿用edit的test_edit_rfp_project_tabs.py文件
   - tests/operate/rfp_management/test_edit_rfp_project_tabs.py
4. 注意测试用例层和page形式写法，做到POM形式，元素方法和定位形式分明
    - page在：pages/operate/rfp_management/edit_rfp_project_page.py
5. 使用参数化的形式提取出来，因为里面的传入项目名可能会改变

## 步骤描述
1. 首先到/home页
2. 点击RFP Management菜单，选择下拉菜单中的：Contracting选项 
3. 然后再Contracting的页面中，需要先选择Started的Tab项
4. 点击搜索栏目的Project填入固定项目名：hyg-测试项目-新加坡酒店类
5. 点击搜索按钮，精准匹配出现的首项
6. 接着在首项目的Operation栏中，存在编辑按钮：Contract Signing，可能存在多个需要精准匹配首个
7. 点击首个编辑按钮：Modify Project
8. 进入到修改项目编辑页
9. 断言LRA和NLRA选项的开关选项，只存在turn on 和 turn off
