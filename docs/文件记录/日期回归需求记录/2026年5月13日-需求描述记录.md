## 需求描述
需求标题：平台和酒店端的价格状态流转
具体描述：
1. 这是一个涉及到多端的登录态情况;
2. 平台端操作酒店归属议价中状态
3. 然后操作酒店端进行提交报价，
4. 最后回到平台端查看是否酒店状态变成修改报价，操作酒店变成中签
2. 实现代码在tests的e2e目录下
   - 目录：tests/e2e/
3. 注意测试用例层和page形式写法，做到POM形式，元素方法和定位形式分明
4. 使用参数化的形式提取出来，因为里面的传入项目名可能会改变

## 步骤描述
1. 首先是平台端的页面，进入到首页工作台页
2. 点击签约管理菜单，选择下拉菜单中的：签约选项 
3. 然后在签约的页面中，需要先选择已启动的Tab项
4. 点击搜索栏目的签约项目-填入固定项目名：自动化测试项目-价格变换
5. 点击搜索按钮，精准匹配出现的首项
6. 点击：去签约按钮，进入去签约地图模式
7. 切换到列表模式
8. 编辑Excel文件，设置酒店为：Shangri-La Hotel Hefei，他的酒店id是：463437
9. 导入签约状态 Excel 文件，将酒店置为指定初始状态：议价中
10. 然后是酒店端，这里我在想考虑新建一个浏览器（毕竟后续还要回到平台端进行）
11. 在酒店端中，同样先到工作台页
12. 点击按钮：报价项目
13. 选择Tab：议价中，这个按钮后面的文字会带上(数字)，需要模糊匹配
14. 在搜索栏目的签约项目栏输入具体的项目：自动化测试项目-价格变换
15. 在搜索栏目的酒店名称栏输入具体的酒店：Shangri-La Hotel Hefei
16. 输入这个酒店后，会出现一个下拉框匹配控件，选中精准第一个匹配
16. 点击搜索按钮
17. 这里可能会出现多个匹配项目，需要出现检索一下页面中，精准匹配上述指定具体项目那一行的Modify Proposal
17. 点击修改报价按钮：Modify Proposal
18. 跳转到报价详情页中，会出现备注留言弹窗，点击关闭按钮
19. 点击提交报价按钮：Submit quotation
20. 出现二次确认弹窗，点击确定按钮：OK
21. 断言是否：操作成功（这里可能会有多个toast，但有一个有就可以了）
22. 需要再次回到平台端的浏览器页面中
23. 在修订报价的价格页签中，先点击该修订报价页签
24. 查看是否存在酒店：Shangri-La Hotel Hefei
25. 点击该酒店中签进行中签操作
26. 断言该酒店是否属于中签页签下

def run(playwright: Playwright) -> None:
    browser = playwright.chromium.launch(headless=False)
    context = browser.new_context()
    page.get_by_role("button", name="Work Table").click()
    page.get_by_role("menuitem", name="Work table").click()
    page.get_by_role("button", name="RFP Management").click()
    page.get_by_role("menuitem", name="Contracting").click()
    page.get_by_role("tab", name="Rebid (16)").click()
    page.locator("#el-id-1583-334").click()
    page.locator("#el-id-1583-359").click()
    page.locator("div").filter(has_text=re.compile(r"^Hotel Name$")).nth(1).click()
    page.locator("#el-id-1583-359").fill("Shangri-La Hotel Hefei")
    page.get_by_role("option", name="Shangri-La Hotel Hefei Hefei").click()
    page.locator("button").click()
    page.get_by_text("Modify Proposal").click()
    page.get_by_role("button", name="Submit quotation").click()
    page.get_by_role("button", name="OK").click()
    page.locator("#message_1").click()
