

# === 1:报价详情增加“内部跟进备注”功能-展开功能 ===
## 需求描述
需求标题：报价详情“内部跟进备注”功能-显示不下的展开功能存在性
具体描述：
1. 使用一个存在指定存在备注信息的酒店项目，进去点击项目酒店，检测备注栏目中显示展开情况;
2. 实现代码在operate的rfp_management目录下，使用已有用例文件：rfp_detailPage_project.py，新构建一个用例方法
   - tests/operate/rfp_management/test_rfp_detailPage_project.py
3. 注意测试用例层和page形式写法，做到POM形式，元素方法和定位形式分明
    - page在：pages/operate/rfp_management/rfp_detailPage_project.py
4. 使用参数化的形式提取出来，因为里面的传入项目名可能会改变

## 步骤描述
1. 首先到/home页
2. 点击RFP Management菜单，选择下拉菜单中的：Contracting选项 
3. 然后再Contracting的页面中，需要先选择Started的Tab项
4. 点击搜索栏目的Project填入固定项目名：自动化测试项目-价格页签固定酒店项
5. 点击搜索按钮，精准匹配出现的首项
6. 接着在首项目的Operation栏中，存在编辑按钮：Contract Signing，可能存在多个需要精准匹配首个
7. 在去签约地图模式下中，选择如新标的页签：New proposal
8. 新标的页签里面可能存在多个酒店，我们可以选择第一个就可以了，参考之前写的
9. 然后在地图的右侧会有酒店详情弹窗，需要点击详情按钮：Detail按钮（待定，先用填充元素），进入详情页
10. 查看备注项目中是否存在展开按钮
11. 存在点击展开后，查看是否存在收起按钮



## 以下是我使用playwright工具录制的，你可以参考一下使用，避免自行定位不到可以考虑使用

def run(playwright: Playwright) -> None:
    browser = playwright.chromium.launch(headless=False)
    context = browser.new_context()
    page.get_by_role("button", name="RFP Management").click()
    page.get_by_role("menuitem", name="Contracting").click()
    page.get_by_role("tab", name="Started", exact=True).click()
    page.locator("#el-id-1615-59").click()
    page.locator("#el-id-1615-59").fill("自动化测试项目-价格页签固定酒店项")
    page.locator(".search > .btn").click()
    page.get_by_text("Contract Signing").first.click()
    page.get_by_text("Mondrian Singapore Duxton Four Star16A DUXTON HILL, Singapore, 089970, Duxton").click()
    with page.expect_popup() as page1_info:
        page.get_by_role("button", name="Detail").click()
    page1 = page1_info.value
    page1.get_by_text("Expand ▼").click()
    page1.get_by_text("Retract ▲").click()

    # ---------------------
    context.close()
    browser.close()
